List of the some tag in html

  1. P Tag --> P tag is used to add paragraph in the html page
  2. A tag --> Anchor tag is used to add hyper link in the page for moving on the another page by clicking on the text or link
  3. Div tag --> div tag can divide page the in multiple division. its a block level element and used as wrap the content.
  4. Aside tag --> aside tag is a semantic tag which is used to define the index on the side of the page
  5. html tag --> html tag is the most important tag that is known as root tag every html tag must contain the root element
